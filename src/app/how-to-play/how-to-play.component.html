How to Play?

STEP #1


Select a mode between Best Response Dynamics questions and Review Flip Cards.

STEP #2


If you selected Best Response Dynamics, choose which category would you like to play then choose which number of scenario would you like to answer.





If you selected Review Flip Cards: You can start learning by clicking the cards.

STEP #3


If you selected Best Response Dynamics: Answer the questions by identifying what the best response to that scenario.



If you selected Review Flip Cards: Now you can read and be familiar with the Kapampangan words that you still don't know.
